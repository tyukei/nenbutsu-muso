# 念仏無双〜修行僧の逆襲〜 音声ファイル設定ガイド

## 必要なフォルダ構成

```
あなたのフォルダ/
  ├─ nenbutsu-musou-with-sound.html  （ゲーム本体）
  └─ sounds/                          （音声フォルダ）
      ├─ bgm.mp3           （ゲーム中のBGM）
      ├─ shoot.mp3         （念仏発射音）
      ├─ hit.mp3           （煩悩撃破音）
      ├─ damage.mp3        （ダメージ音）
      ├─ gameover.mp3      （ゲームオーバー音）
      └─ clear.mp3         （クリア音）
```

## 各音声ファイルの説明

### bgm.mp3（必須）
- **用途**: ゲーム中に流れるBGM
- **推奨**: 読経、和風、神秘的な雰囲気の曲
- **長さ**: 30秒〜2分（ループ再生されます）
- **音量**: 自動で50%に調整されます

### shoot.mp3（必須）
- **用途**: 念仏を発射した時の音
- **推奨**: 鐘の音、シャキーン、リン（仏具）の音
- **長さ**: 0.1〜0.5秒の短い音

### hit.mp3（必須）
- **用途**: 煩悩を撃破した時の音
- **推奨**: 破裂音、消滅音、キラキラ音
- **長さ**: 0.2〜0.8秒

### damage.mp3（必須）
- **用途**: 煩悩を逃してダメージを受けた時
- **推奨**: 重い鐘の音、ドーン、衝撃音
- **長さ**: 0.5〜1.5秒

### gameover.mp3（オプション）
- **用途**: ゲームオーバー時
- **推奨**: 悲しい鐘の音、重低音
- **長さ**: 2〜5秒

### clear.mp3（オプション）
- **用途**: 108の煩悩を全て撃破した時
- **推奨**: 華やかな鐘の音、ファンファーレ風
- **長さ**: 3〜8秒
