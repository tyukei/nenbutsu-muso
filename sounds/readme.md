# 煩悩シューティング 音声ファイル設定ガイド

## 必要なフォルダ構成

現在のプロジェクトでは、以下のような音声ファイル構成になっています。

```
sounds/               （メイン音声フォルダ）
  ├─ bgm.wav          （ゲーム中のメインBGM）
  ├─ shoot.mp3        （読経・弾の発射音）
  ├─ hit.mp3          （煩悩を撃破した時の基本音）
  ├─ hit_bounas.mp3   （連続撃破などのボーナス音）
  ├─ damage.mp3       （被弾した時のダメージ音）
  └─ mika/            （各種ボイスや特殊演出用フォルダ）
      ├─ start.mp4         （「修行の開始」選択時）
      ├─ level1.mp4        （「入門」レベル選択時）
      ├─ level2.mp4        （「中級」レベル選択時）
      ├─ level3.mp4        （「上級」レベル選択時）
      ├─ level4.mp4        （「悪魔」レベル選択時）
      ├─ rokuharamitsu.mp4 （必殺技「六波羅蜜」発動時）
      ├─ clear.mp4         （ゲームクリア時）
      └─ gemover.mp4       （ゲームオーバー時）
```

## 各種音声の役割

### 環境音・効果音
- **bgm.wav**: ゲームプレイ中のBGM。ループ再生されます。必殺技発動時などは一時的に音量が0になります。
- **shoot.mp3**: プレイヤーが攻撃した際の効果音。
- **hit.mp3**: 煩悩に弾が当たったときのヒット音。
- **hit_bounas.mp3**: 特殊な撃破をした際などのボーナス音。
- **damage.mp3**: 敵が画面下部へ到達し、ダメージを受けてしまった時の音。

### mika/ ディレクトリ（ボイス・演出系）
各種の画面遷移やイベント時に再生される音声（.mp4形式）がまとめられています。
- **start.mp4**: タイトル画面で「スタート」を押した後に流れる音声つき待機演出。
- **levelX.mp4**: 各難易度を選択した際に流れるボイスや固有の音。
- **rokuharamitsu.mp4**: 特殊ゲージが貯まった状態で必殺技「煩悩即菩薩（六波羅蜜）」を発動した時に再生される音声。この再生中（3秒間）は他のゲーム音がミュートになり、特別な演出が入ります。
- **clear.mp4 / gemover.mp4**: プレイ終了時の勝敗リザルト画面用音声。
